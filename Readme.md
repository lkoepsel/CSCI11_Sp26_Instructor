# Readme

1. Be sure to **COPY** the week's assignments from this repository to your homework repository (CSCI11_Sp26_Student).

In the Terminal:
```bash
cd CSCI11_Sp26_Instructor

# be sure to change "n" to the current week
# And make sure there is NOT a / after the "n"
cp -r Week_n ../CSCI11_Sp26_Student
```

2. Complete each assignment in the appropriate markdown file.

3. Remember to do the following, or your homework won't be graded:

## Instructions for Updating Homework
   ```bash
   git add -A # to add (stage) all of the changes
   git commit -m "adding my changes for week 5" # to commit your changes
   git push # to push your change to your repository on GitHub
   ```
